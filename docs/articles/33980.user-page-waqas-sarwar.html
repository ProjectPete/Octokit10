User Page: Waqas Sarwar - TechNet Articles - United States (English) - TechNet Wiki

Waqas Sarwar is Infrastructure architect / consultant has over 10 years of experience in Microsoft technologies.  

He has extensive experience in architecture, installation, configuration, and monitoring of almost all version of SharePoint Server (2001, 2003, 2007, 2010, 2013, 2016). Apart from this, he has experience in the Windows Server (2008, 2012), SQL Server (2008, 2012, 2014, 2016), SCOM 2012, Windows (7, 8,10), Metalogix migration tools and DPM. Waqas is an active blogger and contributed a lot in the SharePoint Communities including the SharePoint Social Forum, Technet Wiki, StackExchange SharePoint Forum. He has been awarded Microsoft Community Contributor.
