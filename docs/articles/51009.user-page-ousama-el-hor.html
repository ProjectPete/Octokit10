Ousama EL HOR is a Data & B.I Consultant based in France. He is passionate about SQL Server, Business intelligence, Azure Platform, Power BI, new technologies and more. He loves to share his knowledge, help others and also learn from others as he believes everyone has something which can be learned. In the last years he has been working on multiple MSBI projects using SSIS, SSAS, SSRS with SQL Server and designing, implementing data warehouse projects and administrating of SQL Server.
