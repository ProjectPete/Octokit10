Introduction

Emiliano Musso works in IT since 2003 as Software Developer for a company in Northern Italy. He was awarded in 2014 with his first Microsoft MVP status (renewed in 2015, 2016, 2017-2018), for Visual Basic .NET expertise (.NET in 2015, and Visual Studio and Development Technology since 2016), and MCC (Microsoft Community Contributor) status in 2015. His main interests and skills are in .NET Framework, Visual Basic, Visual C# and SQL Server.
