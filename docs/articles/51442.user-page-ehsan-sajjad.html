 Introduction


Ehsan Sajjad is a Microsoft Certified Professional, TechNet Wiki Ninja, CodeProject Most Valuable Professional, Microsoft Certified Solutions Developer and Architect with over 5 years of experience in Microsoft Technologies.

Ehsan Sajjad is from Islamabad, Pakistan. He is an active contributor on different online communities which include Microsoft TechNet Wiki, MSDN forums, StackOverflow, CodeProject, CSharpCorner. He was awarded CodeProject Most Valuable Professional award for year 2018 and is the 4th Top User on StackOverflow from Pakistan and overall in Top 0.26% users. He has done Bachelors in Computer Science in 2012 and since after that working as a Software Developer in .NET Technologies.

His work experience includes different tools and Technologies including C#, ASP.NET, ASP.NET MVC, ASP.NET Web API, SQL, JavaScript, JQuery, Entity Framework etc.
