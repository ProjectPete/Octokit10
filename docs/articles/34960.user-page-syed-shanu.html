Introduction

Syed Shanu is a Microsoft MVP for Visual Studio and Development and has more than 10 years of experience in Microsoft Technologies.

Syed Shanu is from Madurai, Tamil Nadu, India. He received his first Microsoft MVP Award for Visual Studio and Development in 2016. He was two times Code Project MVP and CSharpCorner MVP for the year 2015 and 2016. He started programming in C language from his high school at 1999. In 2005 he completed his Masters of Computer Application. He started his working Career in December 2004 with Microsoft Technologies. 

His work experience with Languages and Technologies started from ASP and SQL Server, then VB.NET and C# for PDA Application, Touch Screen Application Development, Desktop Application, ASP.NET Web Application Development, MVC, AngularJs, JQuery, WCF, WEB API and WPF.
